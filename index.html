Agent-Based Room Mapping and Navigation

Using Reinforcement Learning with Real-Time Visualization

Fork on GitHub

Overview

This project simulates a robot vacuum cleaner's behavior using reinforcement learning, specifically a Deep Q-Network (DQN) algorithm. It evaluates and visualizes the agent's performance, showing its ability to clean a room while avoiding obstacles and optimizing its trajectory.

Features

  • Reinforcement Learning Agent: Trained using DQN to learn optimal cleaning paths.
  • Simulated Environment: Dynamic room layouts with dirt, obstacles, and furniture.
  • Real-time Visualization: Visualizes the robot’s actions and performance metrics.
  • Data Storage: Stores simulation results in MongoDB.

Installation

  1. Clone the repository: 
    git clone https://github.com/shuayib-umassd/vacuum-simulator.git
  2. Install dependencies: 
    pip install -r data/requirements.txt
  3. Run the simulation: 
    python simulation.py
  4. Generate video from frames: 
    
cd frames
ffmpeg -framerate 30 -i simulation_frame_%d.png -c:v libx264 -pix_fmt yuv420p simulation.mp4

Libraries Used

  • TensorFlow for DQN algorithm.
  • Pygame for real-time visualization.
  • MongoDB for data storage.
  • FFmpeg for video generation.
